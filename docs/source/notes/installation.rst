Installation
========================

.. note::

   OpenGSL requires `Python 3.7+ <https://www.python.org/>`_

Using Pip
----------------------------------
.. code-block:: bash

    pip install opengsl


Installation for local development
----------------------------------

You can install the latest version of OpenGSL from our github `repository <https://github.com/OpenGSL/OpenGSL>`_.

.. code-block:: bash

    git clone https://github.com/OpenGSL/OpenGSL
    cd opengsl
    pip install -e .
